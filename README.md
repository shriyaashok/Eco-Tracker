Project scaffold for EcoTracker. This repository contains frontend (React), backend (Node/Express/MongoDB) and an optional ML microservice (FastAPI).

Quick start:

1. Backend

```powershell
cd backend
npm install
npm run dev
```

2. Frontend

```powershell
cd frontend
npm install
npm start
```

3. (Optional) ML service

```powershell
cd ml_service
pip install -r requirements.txt
uvicorn api:app --reload --port 8001
```
